# Logica Programacion 3 - Factorial en JavaScript

## Descripción
Este proyecto consiste en crear un programa en JavaScript que calcule el factorial de un número ingresado por el usuario.

El programa:
- Solicita un número mediante prompt.
- Verifica que el dato ingresado sea válido.
- Calcula el factorial usando un ciclo for.
- Muestra el resultado en consola y en pantalla.

- JavaScript
